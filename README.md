# dse-ai

AI-friendly CLI for DSentiment APIs

## Installation

```bash
npm install -g dse-ai
```

## Usage

```bash
dse-ai --help
```

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build
npm run build

# Link globally for testing
npm run link

# Run tests
npm test
```
